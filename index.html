安装前提官方文档

- Node.js
- Git
- hexo
    "$ npm install -g hexo-cli"

在一个文件夹中建立blog需要的各种文件官方文档

- "$ hexo init <folder>"   #初始化
- "$ cd <folder>"          #切换文件夹
- "$ npm install"          #安装

基本指令官方文档

- "$ hexo generate"
- "$ hexo deploy"
- "$ hexo clean"
- "$ hexo server"

更改网站配置官方文档

- language     网站使用的语言
- url     网址     
- root     网站根目录
- theme 当前主题名称
